# UI/UX 및 디자인 시스템 명세서 (UI/UX & Design System Specification)

본 문서는 MyVoca 애플리케이션에 입혀진 고급 디자인 스타일링 원칙, 글래스모피즘 적용 지침, 마이크로 인터랙션 피드백, 그리고 아이콘 에셋 단일 창구화 설계를 명세합니다.

---

## 1. 디자인 테마 및 글래스모피즘 (Glassmorphism Core Guidelines)

MyVoca는 단순하고 밋밋한 정적 AI 템플릿 스타일을 철저히 배제하고, 은은하면서도 미래지향적이고 입체적인 **글래스모피즘(Glassmorphism)** 비주얼 디자인을 전면 채택했습니다.

### 주요 스타일링 사양
- **반투명 블러 백그라운드 (Frosted Glass Effect)**:
  - 컴포넌트 뒷면의 미묘한 흐림 효과를 부여하여 스크롤 시 깊이감과 가독성을 동시 충족합니다.
  - 적용 규격: `backdrop-filter: blur(12px) saturate(180%); background: rgba(255, 255, 255, 0.45);` (다크모드 환경에서는 어두운 HSL 색채계 오버레이 매핑)
- **은은한 테두리 그라디언트 (Subtle Border Highlights)**:
  - 투명한 보더 라인에 얇은 1px 그라데이션을 가미하여 요소의 외곽 윤곽을 정교하고 아름답게 분리합니다.
- **Harmony HSL 컬러 팔레트**:
  - 원색(순수 Red, Blue 등)의 사용을 배제하고 눈의 피로를 최소화하면서도 트렌디한 HSL 조율 파스텔 톤(예: 소프트 바이올렛, 뮤티드 민트, 딥 네이비)을 메인 테마로 혼합 구성합니다.

---

## 2. 마이크로 인터랙션 및 애니메이션 (Micro-Animations & Transitions)

사용자가 애플리케이션의 핵심 기능을 직접 만지고 조작하는 과정에서 생동감을 느끼도록 다양한 인터랙티브 이펙트를 제공합니다.

| 인터랙션 영역 | 애니메이션 물리 가이드라인 및 동작 규칙 | 사용자 경험적 효과 |
| :--- | :--- | :--- |
| **암기 카드 뒤집기 (Card Flip)** | 3D 입체 투영 물리 모델(`transform: rotateY(180deg)`)을 적용하여 탭하는 순간 매끄럽게 카드가 반전되는 회전 속성을 이식합니다. | 단어 인지 학습 몰입도 배가 |
| **타이머 서클 게이지 (CircleTimer)** | CSS 넌블로킹 하드웨어 가속 트랜지션을 사용하여 남은 시간 틱에 맞춰 게이지 테두리가 끊김 없이 균일하게 수축 감소하도록 제어합니다. | 퀴즈 풀이 시간의 긴박감 및 스피디한 연출 |
| **리스트 항목 스와이프 및 스왑** | 학습 정렬 우선순위 스왑 완료 시, 요소가 순식간에 날아와 자신의 바뀐 스케줄 번호 자리에 매끄러운 트랜지션으로 복원 안착합니다. | 데이터 이동에 대한 뛰어난 가시성 제공 |

## 3. 아이콘 에셋 단일 진입점 (`iconList`)

프론트엔드 코드 전반에 중구난방으로 파편화되어 직접 내포되던 거대한 SVG 로우 코드로 인한 의존성 오염 및 빌드 효율성 하락 문제를 영구 해결하기 위해 **`iconList` 단일 진입점 설계**를 구축했습니다.

### 아키텍처 상세
- **위치**: `src/assets/iconList.jsx` (혹은 관련 모듈)
- **운용 지침**:
  - 프로젝트 내에서 사용되는 모든 미려한 테마 아이콘 컴포넌트는 오직 이 단일 파일에 집약 정의됩니다.
  - 개별 UI 컴포넌트 파일 내부에서는 복잡한 인라인 SVG 드로잉 코드를 직접 가질 수 없으며, 반드시 `@/assets/iconList`로부터 필요한 아이콘 모듈을 명시적 임포트하여 사용해야 합니다.
  - 이를 통해 전역 디자인 아이덴티티 및 일관된 테마 색채를 단 한 줄의 중앙 수정으로 일제히 컨트롤할 수 있습니다.

---

## 4. 모바일 퍼스트 반응형 레이아웃 (Responsive Layout System)

- **기본 설계 모드**: 화면 자원이 제한된 모바일 환경에서 한 손 컨트롤이 용이하도록 하단 네비게이션 고정 바와 엄지손가락 터치 세이프 에어리어를 배치한 **모바일 퍼스트(Mobile-First)** 설계를 고수합니다.
- **데스크톱 확장성**: 브라우저 폭이 넓은 PC 환경에서는 콘텐츠 프레임을 중앙 웰컴 그리드로 응축 정렬하고, 오버레이 세이프 영역 외곽의 배경 여백을 아름다운 입체 그라데이션으로 흐리게 감싸 전 시각 구간에서 고품격 룩앤필을 일률 유지합니다.
