# 단어 다운로드 큐 매니저 명세서 (Word Queue Manager Specification)

본 문서는 대용량 영단어 상세 데이터를 UI 프레임 드랍 없이 백그라운드 스레드에서 점진적 및 효율적으로 다운로드하여 로컬 캐시에 병합하는 `WordQueueManager` 싱글톤 관리자와 리액트 브릿지 훅 `useMaster`의 기술 메커니즘을 명세합니다.

---

## 1. WordQueueManager 핵심 다운로드 메커니즘

`src/app/services/WordQueueManager.js`에 구현되어 있으며, 리액트 컴포넌트 생명주기와 완전히 독립되어 백그라운드 단어 다운로드 큐를 통제합니다.

### 1.1 큐 생성 및 캐시 기반 선별 (중복 다운로드 제거)
`setVocaList(vocaList, selectedLabel)`가 실행되면, 매개변수로 수집된 전체 청크 리스트를 평평하게(Flat) 펼친 뒤, 로컬 마스터 캐시(`KEYS.MASTER`)의 존재 여부와 정밀 대조합니다.
- **선별 수집**: 청크 내의 모든 단어 ID가 로컬 캐시에 100% 무결하게 들어 있는 청크는 로드 완료 목록(`loadedChunks`)으로 즉시 분류하여 대기열에서 제외합니다.
- **미캐싱 데이터 수집**: 단 하나라도 로컬 캐시에 존재하지 않는 단어 ID를 포함하고 있는 청크만 다운로드 대기 대상(`chunksToLoad`)으로 선별 수집하여 중복 트래픽을 원천 방지합니다.

### 1.2 대기열 가중치 정렬 및 비집고 들어가기 (Priority Re-sort)
수집된 미캐싱 청크들은 사용자의 학습 맥락에 가장 유기적으로 반응하도록 아래의 순위 가중치에 근거하여 동적으로 재정렬(Re-sort)됩니다.
- **1순위 가중치**: 사용자가 대시보드에서 선택하여 바로 암기를 수행해야 하는 활성 청크 (`selectedLabel`과 청크의 `voca_label`이 일치하는 청크)
- **2순위 가중치**: 스케줄 정렬 순번(`schedule`)이 앞서는 권장 순서
- **동적 격상**: 다운로드 워커가 구동 중인 환경이라도 사용자가 학습 대상을 변경하거나 특정 청크의 암기를 우선 시작하면, `prioritizeChunk(targetId)` 신호를 트리거하여 해당 청크를 대기열 최우선 순위(맨 앞)로 격상시킵니다.

### 1.3 50ms 미세 틱 지연 및 자원 정리 (Cleanup)
- **프레임 드랍 예방**: 다운로드 스레드가 브라우저의 메인 렌더링 스레드를 과도하게 차지하여 UI 렉(Lag)이나 끊김 현상이 일어나는 것을 방지하기 위해, 한 청크의 다운로드가 완료될 때마다 다음 청크로 넘어가기 전 **`50ms` 미세 비동기 틱 지연(Tick Delay)**을 부여해 렌더링 스레드의 숨통을 열어줍니다.
- **워커 소멸**: 대기열에 쌓여 있던 모든 미캐싱 청크의 로드가 완벽히 완수되는 시점 즉시, 비동기 워커 루프를 안전하게 종료하고 `activeWorker = null`로 할당하여 타이머 및 네트워크 스레드 누수(Leak)를 미연에 원천 차단합니다.

## 2. useMaster 브릿지 훅의 옵저버 패턴 연동

`src/app/hooks/useMaster.jsx` 커스텀 훅은 싱글톤으로 도는 `WordQueueManager`의 진행 상황을 전역 리액트 컨텍스트에 넌블로킹으로 전파하는 가교 역할을 합니다.

### 2.1 실시간 구독 및 점진적 병합 (Observer Pattern)
- **옵저버 구독**: 컴포넌트 마운트 시 `wordQueueManager.subscribe()` 콜백을 등록하여 백그라운드에서 특정 청크의 다운로드가 한 번 끝날 때마다 통지 이벤트를 실시간으로 수신합니다.
- **점진적 화면 갱신**: 수신된 단어 데이터는 리액트 상태인 `masterState`에 즉시 누적 병합(`Object.assign`)되어, 사용자가 로딩 바를 통해 실시간 사전 빌드 진행도를 부드럽게 시각적으로 인지할 수 있도록 유도합니다.
- **누수 차단**: 컴포넌트가 언마운트되는 생명주기 종료 단계에서 안전하게 `unsubscribe()`를 실행하여 클라이언트 가비지 컬렉터의 메모리 누수를 예방합니다.
