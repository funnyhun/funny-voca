# Agent Context Anchor - src/app/services/Play

본 문서는 `src/app/services/Play` 도메인의 컴포넌트 및 스타일 코드에 대한 에이전트 전용 컨텍스트 앵커입니다. 에이전트는 이 디렉토리 내의 파일들을 생성, 수정, 리팩토링할 때 아래의 규칙을 반드시 준수해야 합니다.

---

## 1. 디렉토리 구조 및 핵심 역할

```text
Play/
├── AGENT.md                 # 본 문서 (에이전트 제어용)
├── index.js                 # Play 도메인 배럴 파일 (외부 노출 제어)
├── Play.jsx                 # Play 페이지 컴포넌트
├── Play.styles.js           # Play 스타일 파일
├── word.json                # 학습용 기초 데이터셋 백업 (필요 시 참조)
└── [SubComponent]/          # 일반 컴포넌트별 격리 폴더
    ├── index.js             # 배럴 파일
    ├── [SubComponent].jsx      # 리액트 비즈니스 로직 및 마크업
    └── [SubComponent].styles.js # 격리된 styled-components 선언
```

### 각 컴포넌트의 역할
- **Play**: 학습 메인 레이아웃이자 코어 컨테이너. `Outlet`을 사용하여 학습 모드(`Card` 또는 `Quiz`)의 화면 전환 및 로딩 상태를 라우팅 레벨에서 통합 처리합니다.
- **ProgressBar**: 화면 최상단에 마운트되어 전체 학습 대상 단어 대비 완료된 단어의 상태를 미려한 프로그레스 바와 숫자로 사용자에게 직관적으로 피드백합니다.
- **Item**: 단어 카드 학습 모드의 메인 컨트롤러. `useItem` 훅을 바인딩하여 뷰(`Definition` 또는 `Title`)를 오케스트레이션하고 전체 플로우를 총괄합니다.
- **Pannel**: 단어 카드를 뒤집는 액션과 이전/다음 카드로 이동하는 탐색 인터페이스 버튼 세트입니다.
- **Definition**: 단어의 뜻, 품사, 예문 등을 구조화하여 세련되게 보여주는 한글 의미 노출 컴포넌트입니다.
- **Complete**: 세션의 모든 단어 플립 학습이 만료되었을 때, 이를 자축하고 퀴즈 풀기 모드로 즉시 전향하거나 재학습을 할 수 있도록 유도하는 세션 종결 화면입니다.

## 2. 참조 및 의존성 불변 규칙 (Crucial Rules)

1. **상대 참조 금지 (상위 디렉토리 참조 시 절대 경로 사용)**
   - 공용 컴포넌트나 레이아웃, 외부 도메인을 참조할 때는 어떠한 경우에도 `../../common/...` 과 같은 복잡한 상대 경로를 사용할 수 없습니다.
   - 반드시 **절대 경로 별칭 (`@/app/common/...`, `@/common/api/...`, `@/app/services/Play/...`)**을 사용하여 모듈 참조 오류를 차단하십시오.
2. **스타일 파일의 단방향 의존성**
   - `*.styles.js` 파일은 컴포넌트 로직(`*.jsx`) 내에 선언된 훅이나 일반 자바스크립트 변수를 절대 임포트할 수 없습니다.
   - 스타일링에 필요한 가변 속성은 오직 리액트 프롭스(`props`) 및 스타일 전용 임시 프롭(`$prefix` 형식)을 통해서만 전달받아야 합니다.
3. **진입점 최소화 (배럴 패턴)**
   - 도메인 최상위 `index.js`에 명시적으로 노출된 컴포넌트만 외부에서 사용할 수 있습니다. 외부 폴더에서 이 도메인의 서브컴포넌트를 임포트할 때는 반드시 `import { Play, Card } from "@/app/services/Play"` 형태로만 임포트해야 합니다. (별칭 배출 방식으로 `Card`는 실제 `Play/Item`에서 가져옵니다.)
