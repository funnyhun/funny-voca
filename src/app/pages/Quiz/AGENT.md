# Agent Context Anchor - src/app/services/Quiz

본 문서는 `src/app/services/Quiz` 도메인의 컴포넌트 및 스타일 코드에 대한 에이전트 전용 컨텍스트 앵커입니다. 에이전트는 이 디렉토리 내의 파일들을 생성, 수정, 리팩토링할 때 아래의 규칙을 반드시 준수해야 합니다.

---

## 1. 디렉토리 구조 및 핵심 역할

```text
Quiz/
├── AGENT.md                 # 본 문서 (에이전트 제어용)
├── index.js                 # Quiz 도메인 배럴 파일 (외부 노출 제어)
├── utils.js                 # 퀴즈 생성 및 가공 알고리즘 (순수 함수)
├── types/                   # 퀴즈 관련 데이터 구조 스키마 정의
└── [Component]/             # 컴포넌트별 격리 폴더
    ├── index.js             # 배럴 파일
    ├── [Component].jsx      # 리액트 비즈니스 로직 및 마크업
    └── [Component].styles.js # 격리된 styled-components 선언
```

### 각 컴포넌트의 역할
- **QuizSelection**: 학습할 퀴즈 방식(객관식, 주관식, 카드 등)과 문제 수를 선택하는 초기 화면.
- **QuizManager**: 전체 퀴즈 게임의 상태(질문 생성, 점수 관리, 흐름 제어)를 담당하는 오케스트레이터.
- **QuizPannel**: 하나의 퀴즈 문제가 제시되는 실질적인 보드판.
- **Option**: 퀴즈의 개별 선택지 카드 및 정/오답 인터랙션 처리.
- **CircleTimer**: 퀴즈 제한 시간(기본 3초)을 시각적으로 보여주는 SVG 애니메이션 타이머.
- **NumberDisplay**: 제한 시간이 끝난 뒤의 카운트다운 숫자 디스플레이.
- **Complete**: 퀴즈 완료 후 최종 정답 수, 오답 통계 및 보상/피드백을 보여주는 결과 화면.

## 2. 참조 및 의존성 불변 규칙 (Crucial Rules)

1. **상대 참조 금지 (상위 디렉토리 참조 시 절대 경로 사용)**
   - 공용 컴포넌트나 레이아웃, 외부 도메인을 참조할 때는 어떠한 경우에도 `../../common/...` 과 같은 복잡한 상대 경로를 사용할 수 없습니다.
   - 반드시 **절대 경로 별칭 (`@/app/common/...`, `@/common/api/...`)**을 사용하여 모듈 참조 오류를 차단하십시오.
2. **스타일 파일의 단방향 의존성**
   - `*.styles.js` 파일은 컴포넌트 로직(`*.jsx`) 내에 선언된 훅이나 일반 자바스크립트 변수를 절대 임포트할 수 없습니다.
   - 스타일링에 필요한 가변 속성은 오직 리액트 프롭스(`props`) 및 스타일 전용 임시 프롭(`$prefix` 형식)을 통해서만 전달받아야 합니다.
3. **진입점 최소화 (배럴 패턴)**
   - 도메인 최상위 `index.js`에 명시적으로 노출된 컴포넌트만 외부에서 사용할 수 있습니다. 외부 폴더에서 이 도메인의 서브컴포넌트를 임포트할 때는 반드시 `import { Component } from "@/app/services/Quiz"` 형태로만 임포트해야 합니다.
