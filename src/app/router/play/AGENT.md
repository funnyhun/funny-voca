# Play Loader 명세 (`src/router/play/`)

이 디렉토리는 학습(Play) 페이지 진입 시 사용자의 위치를 결정하는 로더 로직을 담당합니다.

## 1. 주요 기능
- 사용자의 인증 상태 및 마지막 학습 위치를 파악하여 적절한 Day로 리다이렉트합니다.

## 2. 리다이렉트 로직
- **Member**: 현재는 기본값으로 `/play/0/card/0`으로 이동합니다.
- **Guest**: 로컬스토리지의 `userData.selected` 값을 참조하여 해당 Day로 이동하며, 범위 초과 시 0으로 폴백합니다.

## 3. 주의 사항
- 이 로더는 직접적인 데이터를 반환하기보다 주로 `redirect`를 수행하는 '내비게이터' 역할을 합니다.
- 실제 데이터 로드는 루트 로더인 `loadUserData`에서 이미 완료되어 `Outlet context`로 전달되므로, 여기서 중복 로드를 피해야 합니다.
